# 💊 Pharmacy Inventory & Sales Analysis

## 📌 Overview
This project analyzes pharmacy inventory and sales performance using Power BI, providing insights into stock levels, supplier performance, and product sales.

## 📊 Key Features
- KPI Tracking:
  - Total Sales
  - Average Unit Price
  - Total Quantity Sold
- Inventory status (Low / Medium / High stock)
- Supplier performance analysis
- Top-selling products

## 🛠️ Tools & Technologies
- Power BI
- DAX
- Power Query
- What-If Analysis

## 📈 Insights
- Identified high-demand products
- Monitored stock levels to prevent shortages
- Evaluated supplier contribution to sales


## 🚀 How to Use
- Open the `.pbix` file using Power BI Desktop
